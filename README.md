About
=====

A HTTP protocol parser.


Installation
============

Lastest Git Revision
--------------------

With CMake:

	$ git clone git://github.com/Neopallium/lua-http-message.git
	$ cd lua-http-message ; mkdir build ; cd build
	$ cmake ..
	$ make
	$ sudo make install


Example usage
=============

	local http_message = require"http_message"
